Troubleshooting Connection Issues:

  1. WiFi Check: Ensure both devices are on the same network
  2. Firewall: Mac System Preferences > Security > Firewall > Allow incoming connections
  3. Try these URLs in order:
    • exp://192.168.4.97:8081
    • exp://localhost:8081 (if testing on simulator)
  4. Restart Expo Go: Force quit and reopen the app
  5. Server Status: Make sure you see "Metro waiting on..." in terminal
